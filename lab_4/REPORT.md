## Формулировка задания
Реализовать программу на C для хранения и поиска элементов с уникальными строковыми ключами и вещественными значениями с использованием сбалансированного дерева.

Ключ - непустая последовательность латинских букв длиной не более 6 символов. Ключи упорядочиваются лексикографически.

Для варианта 5:
1. реализовать красно-черное дерево
2. при печати дерева показывать цвет каждого узла

Программа должна поддерживать операции:
1. добавление узла по ключу со значением
2. удаление узла по ключу
3. печать дерева
4. поиск значения по ключу

Команды задаются в формате:
```text
<номер операции> [<ключ>] [<значение>]
```

## Описание выполненной работы
- `rb_tree.h`, `rb_tree.c`:
  реализованы структуры красно-черного дерева, создание и очистка дерева, проверка ключей, поиск, вставка, удаление, повороты и балансировка после вставки и удаления
- `main.c`:
  реализовано чтение команд из файла, вывод результата, а также ручной режим ввода команд через консоль
- `tests/test.c`:
  реализованы тесты на добавление, поиск, обновление значения, удаление, проверку неверных ключей и работу дерева на примере больше 20 узлов
- `input.txt`:
  добавлен пример входных команд для запуска программы
- `Makefile`:
  реализована сборка основной программы и тестов, запуск из файла, запуск ручного режима и очистка

## Как запустить
Сборка программы:
```bash
make all
```

Запуск программы с командами из `input.txt`:
```bash
make run
```

Запуск ручного режима:
```bash
make run_cli
```

В ручном режиме для выхода можно ввести:
```text
exit
```
или пустую строку.

Сборка тестов:
```bash
make test
```

Запуск тестов:
```bash
make run_test
```

Очистка:
```bash
make clean
```

## Пример работы
Пример входного файла:
```text
1 keym 10.5
1 keya 2.1
1 keyt 7.7
3
4 keya
2 keym
3
```

Пример вывода:
```text
Добавлено: keym 10.5
Добавлено: keya 2.1
Добавлено: keyt 7.7
Дерево:
    (R) keyt 7.7
(B) keym 10.5
    (R) keya 2.1
Найдено: keya 2.1
Удалено: keym
Дерево:
    (R) keyt 7.7
(B) keya 2.1
```

Цвет узла печатается перед ключом:
```text
(B) - черный узел
(R) - красный узел
```
