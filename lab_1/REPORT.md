## Формулировка задания
Реализовать программу на C для дерева `int` с CLI:
1. добавление узла;
2. текстовая визуализация дерева с отступами по глубине;
3. удаление узла;
4. вычисление функции по варианту: определить значение нетерминальной вершины дерева с максимальной глубиной

## Описание выполненной работы
- `tree.h`:
  объявлены структуры `Node` и `Tree`, перечисление `TreeStatus`, прототипы функций
- `tree.c`: сделаны операции добавления, поиска, удаления, печати и очистки дерева
- `main.c`:
  реализовано CLI, ввод команд, вызовы функций дерева, вывод результатов
- `test.c`:
  реализовано 5 тестов основных сценариев
- `Makefile`:
  сборка основной программы и тестов, запуск и очистка

## Как запустить
Сборка программы:
```bash
make all
```

Запуск программы:
```bash
make run
```

Сборка тестов:
```bash
make test
```

Запуск тестов:
```bash
make run_test
```

Очистка:
```bash
make clean
```
