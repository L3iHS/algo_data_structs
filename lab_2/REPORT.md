## Формулировка задания
Реализовать программу на C для сортировки таблицы и двоичного поиска по ключу

Для варианта 5:
1. метод сортировки: простая вставка
2. тип ключа: строковый
3. хранение ключей и данных: раздельное
4. минимальное количество элементов таблицы: 13

Программа должна:
1. читать таблицу из текстового файла
2. выводить исходное состояние таблицы
3. сортировать таблицу
4. выводить результат сортировки
5. выполнять двоичный поиск по введенным ключам
6. проверять работу на трех наборах данных:
   - отсортированный
   - обратный
   - случайный

## Описание выполненной работы
- `records.h`, `records.c`:
  реализованы структура `Records`, чтение таблицы из файла, вывод записей и освобождение памяти
- `sort.h`, `sort.c`:
  реализована сортировка методом простой вставки для строковых ключей с раздельным хранением ключей и данных
- `search.h`, `search.c`:
  реализован двоичный поиск по строковому ключу
- `main.c`:
  реализован запуск программы на трех наборах данных, вывод таблицы до и после сортировки, ввод ключей для поиска
- `create_samples.py`:
  реализована генерация файлов `sorted_samples.txt`, `reversed_samples.txt`, `random_samples.txt`
- `tests/test.c`:
  реализованы простые тесты на чтение, сортировку и двоичный поиск
- `Makefile`:
  реализована сборка основной программы и тестов, запуск и очистка

## Как запустить
Сборка программы:
```bash
make all
```

Запуск программы:
```bash
make run
```

Сборка тестов:
```bash
make test
```

Запуск тестов:
```bash
make run_test
```

Очистка:
```bash
make clean
```

Генерация файлов с данными:
```bash
python3 src/create_samples.py
```

## Формат входных данных
Файл содержит:
1. количество записей
2. далее для каждой записи:
   - ключ
   - данные

Пример:
```text
3
key_1
value_1
key_2
value_2
key_3
value_3
```

## Пример работы
Программа последовательно обрабатывает три файла:
- `data_samples/random_samples.txt`
- `data_samples/sorted_samples.txt`
- `data_samples/reversed_samples.txt`

Для каждого файла:
1. выводится исходная таблица
2. выполняется сортировка
3. выводится отсортированная таблица
4. пользователь вводит ключи для поиска
5. программа выводит найденную запись или сообщение об отсутствии ключа

Пример поиска:
```text
Введите ключ для поиска (Enter или exit - завершить): key_5
Найдено: key = key_5, data = value_5
```
