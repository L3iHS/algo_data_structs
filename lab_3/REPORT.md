## Формулировка задания
Реализовать программу на C для преобразования арифметических выражений с использованием дерева выражения

Для варианта 5:
1. упростить выражение, выполнив возведение чисел в степень с целым показателем
2. преобразование выполнять над деревом выражения
3. выражение сначала переводить в обратную польскую нотацию, затем строить дерево

Программа должна:
1. читать арифметическое выражение в инфиксной форме
2. разбивать выражение на токены
3. переводить выражение в обратную польскую нотацию
4. строить дерево выражения
5. выполнять упрощение выражения по варианту
6. выводить исходное дерево выражения
7. выводить упрощённое дерево и итоговый результат
8. содержать набор тестов

## Описание выполненной работы
- `ast.h`, `ast.c`:
  реализованы структура узла дерева выражения, создание узлов для чисел, переменных и операторов, а также освобождение памяти
- `parser.h`, `parser.c`:
  реализованы токенизация входной строки, перевод выражения в обратную польскую нотацию и построение дерева по ОПН
- `transform.h`, `transform.c`:
  реализовано рекурсивное упрощение дерева и вычисление числовых поддеревьев, включая возведение в степень для варианта 5
- `print.h`, `print.c`:
  реализован вывод дерева выражения с отступами и вывод выражения обратно в инфиксной записи
- `main.c`:
  реализованы ввод выражения, построение дерева, вывод исходного дерева, упрощение, вывод результата и освобождение памяти
- `tests/test.c`:
  реализованы тесты на парсинг и преобразование выражений
- `Makefile`:
  реализована сборка основной программы и тестов, запуск и очистка

## Как запустить
Сборка программы:
```bash
make all
```

Запуск программы:
```bash
make run
```

Сборка тестов:
```bash
make test
```

Запуск тестов:
```bash
make run_test
```

Очистка:
```bash
make clean
```

## Тестирование
В программе реализовано 6 тестов:
1. простая степень `2 ^ 3`
2. выражение `(2 + 3) ^ (1 * 7)`
3. сложное выражение с делением
4. выражение с переменной в основании степени
5. деление с остатком, которое не должно вычисляться
6. степень с отрицательным показателем, которая не должна вычисляться
